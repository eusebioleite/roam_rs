# roam_rs

**roam_rs** displays the size of files and directories.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/eusebioleite/roam_rs.git
cd roam_rs
```

2. Compile:

```bash
cargo build
```

## Use

```bash
./rust_roam -t "C:\Users\Eusébio\Documents"
```

Output:

```
C:\Users\User\Documents -> 15.3GB
├── ProjectA 8.2GB
├── ProjectB 4.5GB
├── Notes 2.1GB
└── todo.txt 512.0KB
```
